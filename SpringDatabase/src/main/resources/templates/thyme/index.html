Thymeleaf

  • Spring Boot에서 많이 사용하는 [View 생성용 템플릿 엔진]
  • 기존의 JSP(EL/JSTL)에서 불필요한 기능을 많이 덜어낸 비슷한 기능들을 제공한다
  • HTML의 형태를 최대한 유지하면서 편리하게 웹 페이지를 작성할 수 있다
  • 기본적으로 HTML의 속성 앞에 th:를 붙이는 방식으로 값을 끌어다 사용한다
  • th:를 붙여서 넣는 값이 기존의 값을 덮어쓰게 된다

Thymeleaf의 리터럴

${}: 변수 표현식

  • EL에서 제공하던 변수 표현식을 사용한다
  • 기본적으로 th:를 붙인 속성 내부에서 동작한다
  • fruit: [[${fruit}]]

@{}: 링크 표현식

  • 원하는 경로를 만들기 위해 사용하는 표현식
  • contextPath가 있는 경우 해당 경로에 contextPath를 추가해준다
  • 절대 경로 사용: [[@{https://naver.com}]]
  • 상대 경로 사용: [[@{./add}]]
  • 도메인 경로 사용: [[@{/employee/list}]]
  • 프로토콜 경로 사용: [[@{//naver.com}]]
  • 쿼리 스트링 파라미터 사용하기: [[@{ /fruit/detail(id=${fruit2.fid}) }]]
  • 쿼리 스트링 여러개 사용하기: [[@{ /fruit/detail(fid=${fruit2.fid},fname=${fruit2.fname}) }]]
  • 경로에 변수 넣기: [[@{ /fruit/{fid}/delete(fid=${fruit2.fid}) }]]
  • 경로에 변수 넣고 쿼리 스트링과 함께 사용: [[@{ /fruit/{fid}/modi(fid=133,fname=grape,fprice=1000) }]]

th:text

해당 요소의 innerText를 채운다

th:utext

해당 요소의 innerHTML을 채운다 (태그 적용)

th:each

  • 타임리프에서 반복문을 돌리기 위해 사용한다
  • 모델에 있는 iterable 객체(배열, 리스트, 셋...)를 사용해야 한다
  • 여러번 반복하고 싶은 태그의 속성에 추가하여 사용한다
  • 값을 두 개 받으면 두 번째 값은 해당 반복문의 상태 객체가 된다
값 index count first last even odd current
[[${ snack }]] [[${ status.index }]] [[${ status.count }]] [[${ status.first }]] [[${ status.last }]] [[${ status.even }]] [[${ status.odd }]] [[${ status.current }]]

  • 필요한 경우가 가끔 있다

th:if, th:unless

  • th:if - 해당 조건이 참인 경우 출력
  • th:unless - 해당 조건이 거짓인 경우 출력
  • if ~ else가 한 묶음으로 적용되지 않고 따로 적용되므로 주의
